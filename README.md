
# Medicine Schedule Website

## Description
A website that allows users to schedule and track their medication intake. Built using **Quasar** and **Firebase**.

## Features
- Medicine scheduling
- Reminder notifications
- Real-time data with Firebase

## Technologies
- **Quasar**
- **Firebase**
- **Vue.js**
